Example Links for Downloading Keyman Keyboards

These are keyboard links for testing 10.x development of Keyman for Android .

Ad-Hoc Keyboard Distribution kmp https://

A Keyman KMP keyboard package may contain multiple JS keyboards. Meta-data inside the package is found in kmp.json.

From a browser, clicking these links should trigger the Keyman mobile app to install the keyboard package. If the browser automatically downloads the kmp file, accessing the kmp download should trigger the Keyman app. After the Keyman app unzips the kmp package, it will display a welcome.htm file with the option to "Install" or "Cancel". If no welcome.htm file exists, the packge ID and version are displayed.

Ad-Hoc Keyboard Distribution json legacy keyman://

The legacy method of ad-hoc keyboard distribution uses the keyman:// protocol. The Keyman mobile app processes the json file and downloads the associated files to install the keyboard.

Note: This method of ad-hoc keyboard distribution will be deprecated in 11.x per GitHub Issue #538
